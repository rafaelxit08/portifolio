Desenvolvimento Web

A disciplina de Desenvolvimento Web envolve aprender a criar aplicações e sites para a internet. Os estudantes começam aprendendo os fundamentos da web, como o funcionamento do protocolo HTTP e a estrutura cliente-servidor. Eles também aprendem HTML, a linguagem de marcação usada para criar a estrutura básica de uma página web, e CSS, que é utilizada para estilizar e formatar o conteúdo HTML. JavaScript é outra parte fundamental do curso, permitindo aos alunos adicionar interatividade e dinamismo às suas páginas web. Eles aprendem sobre variáveis, funções, eventos e manipulação do DOM, que é a representação em forma de árvore da estrutura de uma página web. Frameworks e bibliotecas como React, Angular e Vue.js também são explorados, pois facilitam o desenvolvimento web ao fornecer componentes reutilizáveis e outras funcionalidades avançadas. Além disso, os alunos são introduzidos ao desenvolvimento do lado do servidor, onde aprendem a criar aplicativos e serviços que processam solicitações dos clientes e interagem com bancos de dados. Bancos de dados relacionais e não relacionais são abordados, assim como práticas de segurança web para proteger as aplicações contra ataques comuns. Os alunos também aprendem sobre versionamento de código utilizando sistemas como Git, o que é essencial para colaboração eficiente em projetos de desenvolvimento web. Outros tópicos incluem desenvolvimento responsivo e acessibilidade, garantindo que as aplicações web sejam acessíveis em uma variedade de dispositivos e atendam aos padrões de acessibilidade para usuários com deficiências. No geral, essa disciplina fornece aos alunos as habilidades necessárias para criar aplicações web modernas e funcionais, preparando-os para uma carreira no desenvolvimento web.

Manutenção de Computadores

A disciplina de Manutenção de Computadores aborda os fundamentos e práticas necessárias para manter, diagnosticar e solucionar problemas em computadores. Aqui está uma descrição abrangente: Os alunos começam aprendendo os componentes básicos de um computador, incluindo hardware e software. Eles estudam os princípios de funcionamento dos principais componentes internos, como processador, memória RAM, disco rígido, placa-mãe, placa de vídeo e fonte de alimentação. A parte prática da disciplina envolve aprender a montar e desmontar computadores, bem como a instalação e substituição de componentes de hardware. Os alunos também são introduzidos aos procedimentos de segurança elétrica e física durante a manipulação de hardware. Além disso, os alunos aprendem sobre sistemas operacionais, incluindo instalação, configuração e solução de problemas. Eles exploram diferentes sistemas operacionais, como Windows, macOS e Linux, e aprendem a realizar tarefas como formatação de discos, instalação de drivers e gerenciamento de arquivos. Diagnóstico e resolução de problemas são aspectos fundamentais da disciplina. Os alunos aprendem a identificar e solucionar uma variedade de problemas de hardware e software, como falhas de inicialização, erros de sistema, problemas de rede e malware. A manutenção preventiva também é enfatizada, com os alunos aprendendo a realizar tarefas como limpeza interna de computadores, atualização de firmware e aplicação de patches de segurança. Outros tópicos abordados podem incluir gerenciamento de energia, virtualização, backup e recuperação de dados, e boas práticas de segurança da informação. No geral, essa disciplina prepara os alunos para trabalhar como técnicos de suporte técnico, administradores de sistemas ou profissionais de TI, fornecendo-lhes as habilidades necessárias para manter e solucionar problemas em computadores de forma eficaz.

Rede de Computadores

A disciplina de Redes de Computadores oferece uma compreensão abrangente dos princípios, tecnologias e práticas envolvidas na comunicação e interconexão de sistemas computacionais. Os alunos começam aprendendo os fundamentos das redes de computadores, incluindo os conceitos de comunicação de dados, protocolos de rede, arquiteturas de rede e modelos de referência, como o modelo OSI (Open Systems Interconnection) e o modelo TCP/IP. Esta seção explora os diferentes tipos de dispositivos de rede, como switches e roteadores, e como eles encaminham o tráfego de dados através de uma rede. Os alunos aprendem sobre LANs (Local Area Networks), WANs (Wide Area Networks) e tecnologias de comutação, como Ethernet e VLANs (Virtual Local Area Networks). Os alunos estudam os protocolos de comunicação utilizados na Internet e em outras redes, incluindo TCP (Transmission Control Protocol), IP (Internet Protocol), UDP (User Datagram Protocol), ICMP (Internet Control Message Protocol) e DNS (Domain Name System). Esta seção aborda os conceitos de endereçamento IP, incluindo endereços IPv4 e IPv6, máscaras de sub-rede e como dividir redes em sub-redes para otimizar o uso de endereços IP. Os alunos exploram os protocolos e serviços de aplicação usados para comunicação entre programas em diferentes dispositivos de rede. Isso inclui protocolos como HTTP (Hypertext Transfer Protocol), FTP (File Transfer Protocol), SMTP (Simple Mail Transfer Protocol) e DHCP (Dynamic Host Configuration Protocol). Esta seção se concentra nos protocolos de transporte, como TCP e UDP, que são responsáveis pela entrega confiável e eficiente de dados entre sistemas finais em uma rede. Os alunos estudam os protocolos de roteamento e comutação usados para encaminhar pacotes de dados através de redes. Isso inclui protocolos de roteamento dinâmico, como OSPF (Open Shortest Path First) e BGP (Border Gateway Protocol), e técnicas de roteamento estático. Esta seção aborda os princípios e práticas de segurança de rede, incluindo autenticação, criptografia, firewalls, VPNs (Virtual Private Networks) e detecção de intrusões. Os alunos aprendem sobre ferramentas e técnicas de gerenciamento de redes, incluindo monitoramento de desempenho, configuração de dispositivos de rede, diagnóstico de problemas e políticas de segurança. A disciplina pode incluir uma discussão sobre tendências atuais e tecnologias emergentes em redes de computadores, como SDN (Software-Defined Networking), virtualização de rede e redes definidas por intenção. Essa disciplina é fundamental para qualquer pessoa que trabalhe na área de tecnologia da informação ou que deseje entender como as redes de computadores funcionam. Oferece uma base sólida para projetar, implementar, manter e garantir redes eficientes e seguras.

Sistemas Operacionais

A disciplina de Sistemas Operacionais é essencial para entender o funcionamento dos computadores. Ela explora os princípios, estruturas internas e funcionalidades dos sistemas operacionais, que são o software responsável por gerenciar recursos de hardware e fornecer serviços aos programas de aplicação. Os alunos começam aprendendo sobre a importância histórica e contemporânea dos sistemas operacionais. Eles exploram a evolução desses sistemas ao longo do tempo e como eles se tornaram fundamentais para a computação moderna. Dentro da estrutura dos sistemas operacionais, os alunos mergulham nos componentes principais, como o kernel, que é o núcleo do sistema operacional responsável por interagir diretamente com o hardware. Eles também aprendem sobre o escalonador de processos, gerenciador de memória e sistema de arquivos, compreendendo como esses elementos trabalham juntos para executar tarefas essenciais do sistema. A gestão de processos e threads é um aspecto crucial da disciplina. Os alunos estudam como os sistemas operacionais lidam com a criação, escalonamento, sincronização e término de processos e threads, incluindo os diferentes modelos de programação concorrente e paralela suportados. O gerenciamento de memória é outro tópico importante. Os alunos exploram técnicas como paginação, segmentação e alocação de memória dinâmica, entendendo como o sistema operacional administra eficientemente o espaço de memória disponível. Sistemas de arquivos e entrada/saída também são estudados em detalhes. Os alunos aprendem sobre a organização dos dados em sistemas de arquivos, bem como as técnicas utilizadas para gerenciar dispositivos de entrada/saída, como controleadores de dispositivos e buffers de E/S. Além disso, a disciplina aborda tópicos avançados, como segurança e proteção, sistemas distribuídos e multiprocessamento, virtualização e contêineres. Os alunos também podem discutir tendências atuais e desafios emergentes, como computação em nuvem, segurança cibernética e privacidade de dados. Em resumo, a disciplina de Sistemas Operacionais é fundamental para qualquer pessoa que deseje entender o funcionamento interno dos computadores e dispositivos modernos. Ela fornece uma base sólida para o desenvolvimento de software, administração de sistemas e pesquisa em computação.

Linguagem de Programação

A disciplina de Linguagem de Programação é projetada para proporcionar aos alunos uma compreensão abrangente das linguagens utilizadas para criar software. Aqui está uma descrição detalhada: Os alunos começam explorando os conceitos fundamentais da programação, incluindo variáveis, tipos de dados, estruturas de controle (como loops e condicionais) e funções. Eles aprendem como esses elementos são utilizados para resolver problemas e criar algoritmos. Dentro do estudo das linguagens de programação, os alunos são apresentados a diferentes paradigmas de programação, como programação imperativa, orientada a objetos, funcional e lógica. Eles exploram as características distintivas de cada paradigma e como aplicá-los em diferentes contextos de programação. Os alunos aprendem sobre sintaxe e semântica das linguagens de programação, ou seja, as regras para escrever código válido e significativo. Eles praticam a escrita de programas simples e complexos em diversas linguagens, a fim de adquirir proficiência na implementação de algoritmos e estruturas de dados. Além disso, a disciplina aborda tópicos avançados, como manipulação de arquivos, gerenciamento de memória, tratamento de exceções e programação concorrente. Os alunos aprendem técnicas e boas práticas para escrever código robusto, eficiente e fácil de manter. A disciplina também pode incluir uma introdução ao desenvolvimento de software orientado a objetos, com foco em conceitos como encapsulamento, herança e polimorfismo. Os alunos aprendem a projetar e implementar classes e objetos em linguagens que suportam esse paradigma, como Java ou C++. Além disso, os alunos podem ser introduzidos a linguagens específicas para domínio, como linguagens de script, linguagens de consulta de banco de dados ou linguagens de marcação. Eles aprendem como essas linguagens são usadas em diferentes contextos e como integrá-las em seus projetos de software. No geral, a disciplina de Linguagem de Programação é essencial para qualquer pessoa que deseje se tornar um desenvolvedor de software. Ela fornece uma base sólida de conhecimento e habilidades necessárias para escrever código eficaz em uma variedade de linguagens e paradigmas de programação.

Segurança de Sistemas

A disciplina de Segurança de Sistemas é projetada para fornecer aos alunos uma compreensão abrangente dos princípios, técnicas e práticas utilizadas para proteger sistemas de informação contra ameaças cibernéticas. Aqui está uma descrição detalhada: Os alunos começam aprendendo sobre os fundamentos da segurança da informação, incluindo os princípios de confidencialidade, integridade e disponibilidade dos dados. Eles exploram os diferentes tipos de ameaças cibernéticas, como malware, ataques de negação de serviço (DDoS), phishing, engenharia social e violações de dados. Dentro da disciplina, os alunos estudam as políticas, padrões e regulamentações relevantes para a segurança da informação, como GDPR, HIPAA e PCI DSS. Eles aprendem sobre a importância da conformidade regulatória e como implementar controles de segurança para cumprir essas exigências. Os alunos exploram as camadas de segurança em um sistema de informação, incluindo segurança de rede, segurança de sistemas operacionais, segurança de aplicativos e segurança física. Eles aprendem sobre as vulnerabilidades comuns em cada uma dessas áreas e as medidas de proteção apropriadas para mitigar essas ameaças. Dentro da disciplina, os alunos também estudam técnicas de criptografia e criptografia, incluindo algoritmos de criptografia simétrica e assimétrica, funções hash e protocolos de segurança, como SSL/TLS. Eles aprendem como aplicar essas técnicas para proteger a comunicação de dados e garantir a confidencialidade e integridade das informações transmitidas pela rede. A disciplina aborda tópicos avançados em segurança cibernética, como detecção e resposta a incidentes, análise forense digital, testes de penetração e segurança em nuvem. Os alunos aprendem técnicas e ferramentas para identificar, responder e recuperar de violações de segurança de forma eficaz. Além disso, os alunos estudam boas práticas de segurança, como gerenciamento de identidade e acesso, controle de acesso baseado em função (RBAC), políticas de segurança de senha e conscientização do usuário. Eles aprendem como educar os usuários finais sobre práticas seguras de computação e como criar uma cultura de segurança dentro de uma organização. No geral, a disciplina de Segurança de Sistemas é essencial para qualquer pessoa que trabalhe na área de tecnologia da informação ou que deseje proteger sistemas de informação contra ameaças cibernéticas. Ela fornece uma base sólida de conhecimento e habilidades necessárias para desenvolver, implementar e manter medidas eficazes de segurança da informação.
